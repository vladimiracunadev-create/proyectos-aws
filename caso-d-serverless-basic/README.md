# Caso D: Serverless Básico (Lambda + API Gateway) ⚡

> **Nivel 3**: Lógica Dinámica

## Objetivo
Añadir capacidades de Backend sin administrar servidores. Aprenderás a crear APIs que escalan automáticamente y solo cobran por ejecución.

## Stack Tecnológico
- **AWS Lambda** (Node.js)
- **Amazon API Gateway** (HTTP API)
- **AWS IAM** (Roles y Permisos Lambda)

## Escenario
Un botón en la web estática que solicita un saludo personalizado al backend. La Lambda procesa la solicitud y devuelve una respuesta JSON.
