Este caso implementa los tres pilares de la observabilidad en AWS como código: métricas custom en CloudWatch, trazas distribuidas con X-Ray y un dashboard + alarmas definidos directamente en la plantilla SAM/CloudFormation.

SAM / CloudFormation Python 3.12 X-Ray Activo Dashboard IaC 2 Alarmas

Los tres pilares

📊 Métricas

CloudWatch: invocaciones, errores, latencia p99 y métricas custom de negocio en namespace CasoH.

📋 Logs

CloudWatch Logs con salida JSON estructurada. Queries con Logs Insights para analisis rapido de incidentes.

🔗 Trazas

AWS X-Ray activado por SAM Globals. Service map, timeline y subsegmentos por invocacion sin codigo extra.
